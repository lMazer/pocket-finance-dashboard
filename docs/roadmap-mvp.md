# Roadmap MVP (sprints)

## Sprint 1 - Base tecnica e ambiente

- Stack Docker dev com auto-reload (`db`, `backend`, `frontend`)
- Autenticacao backend + Swagger + healthchecks
- Shell/layout inicial Angular
- CI base (backend + frontend build)

## Sprint 2 - Fluxos core financeiros

- Login JWT no frontend
- Categorias CRUD
- Transacoes CRUD + filtros + paginacao
- Dashboard com dados reais
- Budgets/metas por categoria

## Sprint 3 - UX, CSV e qualidade

- Importacao/exportacao CSV
- Refresh token e bootstrap de sessao
- Melhorias de responsividade e acessibilidade
- Scripts de smoke/local DX
- Testes unitarios frontend e backend (camadas criticas)

## Sprint 4 - Docs, release e hardening

- README operacional completo
- Templates de issue e fluxo de contribuicao
- Release Please (versionamento automatico)
- Checklists de release/aprovacao
- Polimento final de UX e backlog p2/p3

## Critério de conclusão do MVP

- Login e sessao funcionando
- CRUD de categorias e transacoes funcionando
- Dashboard e budgets consumindo dados reais
- CSV import/export funcional
- CI verde
- Documentacao operacional minima publicada
